# Secure-Login-System
A secure login web application developed using Flask, SQLite, and bcrypt. The application allows users to register, log in, and securely manage sessions.
# Features
1.User Registration
2.Secure Login
3.Password Hashing (bcrypt)
4.Session Management
5.Logout Feature
6.SQL Injection Protection
# Technologies Used
1.Python
2.Flask
3.SQLite
4.bcrypt
# Installation
pip install flask bcrypt
Run
python app.py
